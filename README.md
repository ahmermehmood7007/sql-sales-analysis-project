# SQL Sales Analysis Project

## Overview
This project is a comprehensive **SQL sales analysis** exercise, covering fundamentals to advanced queries. It includes real-world scenarios for analyzing sales, customers, orders, and products using SQL.

The project is structured to help learners **practice SQL joins, aggregates, filtering, and reporting** while building a complete sales analysis project.

## Key Learning Areas

- Basic SQL queries: `SELECT`, `WHERE`, `ORDER BY`, `LIMIT`  
- Aggregate functions: `SUM`, `COUNT`, `AVG`, `MAX`, `MIN`  
- `GROUP BY` and `HAVING` clauses  
- Joins: `INNER JOIN`, `LEFT JOIN`, `RIGHT JOIN`, `FULL OUTER JOIN`  
- Complex queries involving multiple tables  
- Filtering data using `AND`, `OR`, `NOT`  
- Real-world sales data analysis and reporting
